# react-native-fingerprint-scanner

A library to connect to fingerprint scanner

## Installation

```sh
npm install react-native-fingerprint-scanner
```

## Usage


```js
import { multiply } from 'react-native-fingerprint-scanner';

// ...

const result = multiply(3, 7);
```
